# ServiceSide Sites

A clean, mobile-friendly one-page static website for **ServiceSide Sites**.

## Files

- `index.html` contains the page content and semantic structure.
- `styles.css` contains the mobile-first layout, colors, cards, buttons, and responsive design.
- `script.js` keeps the footer year current.
- `README.md` explains how to edit and preview the site.

## Preview

Open `index.html` in a browser. No build step or server is required.

## Editing Notes

- Update the brand colors in the `:root` section at the top of `styles.css`.
- The contact form is connected to Formspree at `https://formspree.io/f/xqevqdpw`.
- Replace the sample cards in the before-and-after section once you have real examples.
- Update `info@servicesidesites.com` if you choose a different email address.
